# Profile Sharing Permissions

This document describes the permission levels available when sharing a profile with contributors (Databricks users and groups).

## Overview

Profiles can be shared with Databricks Unity Catalog identities (users and groups) at three permission levels:
- **CAN VIEW** – Read-only access
- **CAN EDIT** – Modify profiles and content
- **CAN MANAGE** – Full administrative control

## Profile ACLs

| Ability | NO PERMISSIONS | CAN VIEW | CAN EDIT | CAN MANAGE |
|---------|----------------|----------|----------|------------|
| **Profile Access** |
| See profile in list | | ✓ | ✓ | ✓ |
| View profile configuration | | ✓ | ✓ | ✓ |
| Set profile as personal default | | ✓ | ✓ | ✓ |
| Edit profile configuration (includes contributor management) | | | ✓ | ✓ |
| Delete profile | | | | ✓ |
| **Sessions** |
| View all sessions under profile | | ✓ | ✓ | ✓ |
| Create new sessions | | ✓ | ✓ | ✓ |
| Delete sessions | | | | ✓ |
| **Slides** |
| View slides in sessions | | ✓ | ✓ | ✓ |
| Edit slides | | | ✓ | ✓ |
| Delete slides | | | | ✓ |
| Regenerate slides | | | ✓ | ✓ |

## Permission Level Descriptions

### CAN VIEW

**Ideal for:** Users who need to consume content generated by a profile without modifying it.

Grants:
- View the profile's configuration settings
- Set the profile as their personal default for slide generation
- Access all sessions and slides created under the profile
- Create new sessions to generate slides using the profile

Restrictions:
- Cannot modify profile configuration (Genie spaces, prompts, styles)
- Cannot edit or delete existing slides
- Cannot delete sessions
- Cannot manage contributors

### CAN EDIT

**Ideal for:** Team members who actively collaborate on presentations and need to refine output.

Grants everything in CAN VIEW, plus:
- Edit profile configuration (Genie spaces, deck prompts, slide styles, AI infrastructure)
- Manage contributors (add, remove, modify permissions)
- Edit existing slides (content, layout, formatting)
- Regenerate slides with new prompts

Restrictions:
- Cannot delete the profile
- Cannot delete sessions
- Cannot delete slides

### CAN MANAGE

**Ideal for:** Profile administrators and team leads who need full control.

Grants everything in CAN EDIT, plus:
- Delete the profile entirely
- Delete any session under the profile
- Delete slides
- Transfer ownership (by adding another user with CAN MANAGE)

## Notes

1. **Profile Owner**: The user who creates a profile automatically has CAN MANAGE permissions and cannot be removed.

2. **Group Permissions**: When a group is granted access, all members of that group inherit the permission level.

3. **Permission Inheritance**: Higher permission levels include all abilities of lower levels.

4. **Personal Default**: Any user with at least CAN VIEW can set a profile as their personal default, but this only affects their own user experience.
